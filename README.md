# Customer Churn Prediction System

## Overview
Predicts customer churn for a telecom company using machine learning.
Built with Python, Scikit-learn, and deployed as a Streamlit web application.

## Problem Statement
Customer churn costs telecom companies significant revenue. 
This project identifies customers likely to churn so retention 
teams can intervene proactively.

## Tech Stack
- Python, Pandas, NumPy
- Scikit-Learn (Logistic Regression, Decision Tree, Random Forest)
- Matplotlib, Seaborn
- Streamlit

## Results
- Best Model: Random Forest
- ROC-AUC Score: 0.82
- Accuracy: 0.79

## Key Features
- Full EDA with visualizations
- Class imbalance handling
- Churn risk segmentation (High/Medium/Low)
- Interactive Streamlit deployment

## How to Run
1. Clone the repo
2. Install requirements: pip install -r requirements.txt
3. Run: streamlit run app.py
